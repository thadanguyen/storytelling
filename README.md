# Storytelling Skills

This repository contains installable Codex skills for communication planning and storytelling work.

## Skills

- `storytelling-plan`: plan a talk, presentation, pitch, meeting, or deck before making slides

## Install

Install a single skill from this repository with the Codex skill installer:

```sh
install-skill-from-github.py --repo <owner>/storytelling --path skills/storytelling-plan
```

You can add more storytelling-related skills later under `skills/<skill-name>/`.
